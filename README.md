# Calculator

A responsive calculator web application developed as part of the CodeAlpha Frontend Development Internship.
## Features

- Addition, subtraction, multiplication, and division
- Real-time calculation display
- Clear button to reset calculations
- Backspace button to delete the last character
- Responsive design for different screen sizes
- User-friendly interface

## Setup

1. Clone the repository:

```bash
git clone https://github.com/yagyaraghav61/CodeAlpha_Calculator.git

```
2. Open the project folder:
```bash
cd CodeAlpha_Calculator

```
3. Run the application:
   
- Open index.html in your browser

Functionality
- Enter numbers using calculator buttons
- Select an arithmetic operator
- Press = to get the result
- Use C to clear the display
- Use ⌫ to remove the last entered character
